🤖 AI Agents Web Search & Browse API (LLMs, LangChain, CrewAI)

O RusyaSearch 2.0 oferece endpoints REST e Python SDK otimizados para Agentes de IA. Eles pesquisam no Google/Web, entram em múltiplas páginas e retornam tudo em Markdown limpo e otimizado para o orçamento de tokens do LLM.

🌐 Navegação Avançada para Agentes (`POST /api/v1/agent/browse`) — Suporta renderização JS/Headless, simulação de login/cookies, clique em botões, scroll automático e captura de Screenshot:

🕷️ Crawl de Site Inteiro (`POST /api/v1/agent/crawl`) — Suporta limite de profundidade, paralelismo assíncrono, sitemaps e robots.txt: 
# Exemplo Python SDK para Crawl Profundo
from agent_client import RusyaAgent

agent = RusyaAgent("http://localhost:8080/api/v1/agent")

status = agent.crawl(
    seed_url="https://fastapi.tiangolo.com",
    max_pages=100,
    max_depth=3,
    concurrency=4,
    respect_robots=True
)
print(status)
Exemplo de Uso via Python SDK (agent_client.py): 
from agent_client import RusyaAgent

agent = RusyaAgent("http://localhost:8080/api/v1/agent")

# 1. Busca rápida na web
res = agent.search("lançamento Python 3.13", max_results=5)

# 2. Navegar em uma página e pegar Markdown otimizado para tokens
page = agent.browse("https://pt.wikipedia.org/wiki/Python", max_chars=8000)
print(page["markdown"])

# 3. SUPER PESQUISA (Pesquisa + Navega em N páginas simultaneamente)
dossie = agent.research("tendências de IA agentes autônomos 2026", browse_top_n=3)
print(dossie["consolidated_dossier_markdown"])

Exemplos cURL para Agentes HTTP: 
# Pesquisar na Web

  -H "Content-Type: application/json" \
  -d '{"query": "agentes autônomos LLM", "max_results": 5}'

# Entrar na página e extrair Markdown limpo
curl -X POST http://localhost:8080/api/v1/agent/browse \
  -H "Content-Type: application/json" \
  -d '{"url": "https://pt.wikipedia.org/wiki/Inteligência_artificial", "max_chars": 6000}'
